

**Setting User Name and Email:**
1. To set the user name: `git config --global user.name "user name"`
2. To set the email: `git config --global user.email "emailid"`

**Editing User Name and Email:**
1. To edit the user name and email in a text editor: `git config --global --edit`
   - Exit the editor by pressing Esc, then typing `:wq` and hitting Enter.

**Checking User Name and Email:**
1. To check the user name: `git config --global user.name`
2. To check the email: `git config --global user.email`

**Changing Directory:**
1. To change the directory to a different drive: `cd "C:/folder name"`

**Working with Git:**
1. To initialize Git for a folder: `git init`
2. To check the status of modified files: `git status`
3. To add files to the staging area: `git add file_name_with_file_type`
   - To add all files: `git add .`
4. To commit changes to the repository: `git commit -m "commit message"`
5. To view the commit history: `git log`

**Working with Branches:**
1. To create a new branch: `git branch branch_name`
2. To switch to a branch: `git checkout branch_name`
   - To create and switch to a branch simultaneously: `git checkout -b branch_name`
3. To merge a branch into the current branch: `git merge branch_name`

**Ignoring Files/Folders:**
1. Create a `.gitignore` file in the root of your repository.
2. Add file/folder names to `.gitignore` to exclude them from tracking.

**Pushing to Git:**
1. To add a remote repository: `git remote add origin remote_url`
   - You can use `origin` or another name for the remote.
2. To set the default branch (e.g., from "master" to "main"): `git branch -M main`
3. To push the branch to the remote repository: `git push -u origin branch_name`

**Other Useful Commands:**
1. `ls`: List files and directories.
2. `mkdir folder_name`: Create a new folder/directory.
3. `rm -rf file_name`: Remove a file.
4. `ls -a`: List all files, including hidden files.
5. `ls .git`: List files and directories inside the `.git` folder.
6. `touch file_name.txt`: Create a new file.

Please note that some commands may vary depending on the operating system and Git configuration.
